# Stock Lab Framework

## 目标

建立一个可扩展、可解释、面向多市场的 **股票研究 + 模拟交易 + 看板可视化框架**。

## 公开版边界

本公开仓库只保留：

- 多市场账户结构
- 行情 / 外部数据接入框架
- 风控与纪律执行层
- 状态文件 schema
- 桌面端与移动端看板
- 定时调度脚本

不保留：

- 作者个人持仓
- 作者观察股池
- 复盘日志
- 私有 API key
- 可直接照搬的个人选股模板

## 系统主流程

1. 拉取行情和外部数据
2. 更新账户、持仓与风险暴露
3. 执行强制纪律动作
4. 生成候选与审议结果
5. 输出统一状态文件
6. 前端看板读取状态并渲染

## 核心模块

- `core/config.py`：配置与模板默认值
- `core/market_data.py`：多数据源接入层
- `core/strategy.py`：候选构建 / 评分 / 审议 / 进场
- `core/discipline.py`：止损 / 止盈 / 风控 / 退出
- `core/state_builder.py`：状态聚合与 schema 输出
- `static/`：桌面端与移动端 UI

## 可扩展点

你可以基于这个框架替换：

- watchlist 来源
- 主题分类
- 候选评分器
- 风险阈值
- 调仓与退出规则
- 前端展示方式

## 使用原则

- 先把它当成研究 / 模拟工具
- 再逐步接自己的策略
- 最后才考虑是否对接实盘
